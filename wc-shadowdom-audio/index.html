<audio> shadow tree

<audio> и некоторые другие элементы в браузере представлены с помощью Shadow Tree.

Чтобы заглянуть "под капот" таким элементам, в настройках Chrome DevTools, вкладка General, раздел Elements, нужно поставить галочку Show Shadow DOM.

Тогда в закладке Elements в DevTools у <audio> появится потомок '#shadow-root (user-agent)'. Это и есть Shadow DOM:

Используйте псевдо-селекторы, чтобы изменить представление браузерных элементов, использующих shadow DOM. К таким элементам относятся <input type="range">, <video> и другие. Например, чтобы сделать панель <audio> зелёной, добавьте это в стили: audio::-webkit-media-controls-panel { background-color: green; }
