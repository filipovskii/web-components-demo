<audio> и некоторые другие элементы в браузере представлены с помощью Shadow DOM.

Чтобы заглянуть "под капот" таким элементам, в настройках Chrome DevTools, вкладка General, раздел Elements нужно поставить галочку Show Shadow DOM.

Тогда в разделе Elements в DevTools у <audio> появится потомок '#shadow-root (user-agent)':

Используйте псевдо-селекторы, чтобы изменить представление элементов Shadow DOM. Например, чтобы сделать панель <audio> зелёной, добавьте это в стили: audio::-webkit-media-controls-panel { background-color: green; }
