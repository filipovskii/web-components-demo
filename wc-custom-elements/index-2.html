1. Зелёный свет

Стили, которые определёны в Shadow Tree распространяются только внутри этого дерева. На странице есть и другие divы, но только здесь он зелёный.

var host = document.querySelector('.shadow-host'), template = document.querySelector('#tmpl-green'), shadow = host.createShadowRoot(); shadow.appendChild(template.content);
